USA Property Price Visualization and Prediction

Yupeng Tang

Georgia Institute of Technology

Motivation

Problem: The real estate market involves complex datasets influenced by numerous factors, making it challenging to analyze using traditional tools. These tools often lack intuitive visualizations, predictive analytics, the ability to provide real-time insights, and the scalibity for large datasets, which limits their effectiveness and accessibility.
Why is it important?
  • Understanding and analyzing real estate data is crucial for informed decision-making by buyers, sellers, and investors.
  • By addressing the limitations of traditional methods, our solution integrates interactive visualizations, machine learning models, and large language models to enhance: scalability, interactivity, accessibility, and predictive power.
  • These advancements make complex data accessible to non-experts, enabling natural language interactions and real-time summarization of visualizations.

Our Approaches

Key Components:
  • Interactive visualizations combining state/Zip Code level maps with multi-dimensional view.
  • Machine learning models to provide price prediction.
  • Large language models for real-time insights based on visualizations.
How do they work?
  • Users can explore data through interactive choropleth maps and multi-dimensional plot.
  • Multiple ML models are trained help forecast property prices based on user-selected parameters.
  • LLMs summarize complex visualizations into natural language, making insights accessible to non-experts.
Intuition Behind Our Approach:
  • Interactivity enables users to uncover patterns and correlations across regions and attributes.
  • Predictive Analytics empower users with actionable insights, allowing them to estimate property prices based on multiple factors.
  • Natural Language Summarization ensures accessibility for non-experts, bridging the gap between complex data and user understanding.
Innovation:
  • Use data reduction techniques to address the performance issue for large datasets.
  • Implement drag-and-drop interaction for flexible exploration.
  • Incoporate multi-factor machine learning models for prediction.
  • Provide LLM-driven insights based on visualization by using the GPT-4o API.

Data

Source: Downloaded from the popular real estate dataset on Kaggle.
Characteristics:
  • Coverage: USA nation wide listings of property.
  • Number of Records: Over 2.2 million rows of data.
  • Features: Contain 10 features, such as price, zipcode, number of bedrooms, size of lot, house size, etc.
  • Processing: Spark-based cleaning and aggregation.
Data Cleaning:
  • Deleted features with >50% missing values.
  • Logarithmic transform on target variable price for symmetry and wider distribution.
  • Encoded categorical variables using numerical label encoding.
Machine Learning:
  • Use cross-validation for model evaluation.
  • Hyperparameter tuning with grid search.
  • Feature importance analysis to identify key predictors.

Experiments and Results

Evaluation Approach:
  • Training/testing split validation
  • Metrics: R-square, RMSE, Pearson R
  • Feature importance via SHAP values
  • User interface testing with GT students
Model R square RMSE
Random Forest 0.81 0.12
3-layer Neural Network 0.77 0.14
Linear Regression 0.68 0.19
Key Findings:
  • Random Forest performed best with R² of 0.81
  • Feature importance: house size, bathrooms, acre lot size
  • Better accuracy on lower-priced properties
  • Users are aware of all the features of visualization tool and find it very easy to use.
  • Users think highly of the interactions and LLMs integration.
Comparison to Other Methods:
  • Our R² of 0.81 comparable to benchmark of 0.9 in Ho's Random Forest study.
  • Achieved competitive results with simpler, more interpretable models.
  • Our system enhances scalability, interactivity, accessibility compared to current practices.

Interactive Visualization Interface

  • Upper Left - USA Choropleth Map: Visualizes state-level data with the flexibility to toggle between median property prices and the number of listings.
  • Upper Right - ZIP code-level Choropleth Map: When a state is dragged from the USA choropleth map, a detailed ZIP code-level choropleth map for the selected state is generated.
  • Lower Right - Multi-Dimensional Plot: Corresponding to the ZIP code-level map, this plot help users identify patterns, relationships, and variations among ZIP codes with highlight color.
  • Lower Left - AI Insights Panel: Leverages GPT-4o to generate natural language summaries of the three visualizations.
